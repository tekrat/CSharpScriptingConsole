# C# Scripting Console

## GNU GENERAL PUBLIC LICENSE -  Version 3, 29 June 2007

## Created by: Ervin Kosch

A full-featured interactive C# REPL with:
- Multi-line input
- #load "script.csx"
- Command history (↑/↓)
- Ctrl+R reverse search
- Live syntax highlighting
- Full Roslyn C# 12+ support

Run with: 'CSharpScriptingConsole.exe'
Load a script: 'CSharpScriptingConsole MyScript.csx'
